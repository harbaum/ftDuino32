ftDuino32 - ESP32 based Arduino for fischertechnik

The ftDuino32 is the big brother of the ftDuino. It comes in the same form factor and is mechanically fully compatible with the ftDuino.

The ftDuino32 is still under development. Its expected to become available in 2023 in the same shops that also sell the ftDuino.


The ftDuino32

The main features of the ftDuino32 are:

  • ESP32 based
    • 240 Mhz dual core
    • built-in WiFi and Bluetooth
    • 8 MB RAM
    • 16 MB flash memory
    • color TFT display
  • fischertechnik compatibility
    • Works with all modern 9V fischertechnik power supplies
    • 8 analog 9V fischertechnik compatible inputs I1-I8
    • 4 analog 9V fischertechnik compatible outputs O1-O8
      • Can be used as four motor outputs M1-M4
    • 4 fast counter inputs
    • fischertechnik TX and ftDuino compatible 5V IÂ²C
  • Preinstalled MicroPython
    • Micropython based user interface
    • Full compatibility with all major MicroPython IDEs
    • WiFi and Bluetooth examples
    • Demos and examples for fischertechnik connectivity
