MicroPython/ESP32 for fischertechnik

The ftDuino32 pairs the powerful ESP32 microcontroller with the versatile fischertechnik construction toy.

The ftDuino32 is the big brother of the ftDuino. It comes in the same form factor and is mechanically and electically compatible with the ftDuino.

The ftDuino32 is currently under development. Its expected to become available in 2023 at the same places that also offer the ftDuino.

Specifications

  • ESP32, 240 Mhz dual core
  • Built-in USB, WiFi and Bluetooth
  • 8 MB RAM
  • 16 MB flash memory
  • Color TFT display 160x80
  • Works with all modern 9V fischertechnik power supplies
  • 8 analog 9V fischertechnik compatible inputs I1-I8
  • 8 analog 9V fischertechnik compatible outputs O1-O8
